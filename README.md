# Superstore Data Analysis 📊

## 📌 Project Overview
This project focuses on analyzing retail sales data from a Superstore dataset to uncover key business insights. The analysis includes data cleaning, transformation, and visualization using both Python and Power BI to understand sales performance, profitability, and customer behavior.

## 🎯 Objectives
- Analyze sales and profit trends
- Identify top-performing regions and categories
- Detect loss-making products
- Understand monthly and yearly sales patterns

## 🛠 Tools & Technologies
- Python (Pandas, Matplotlib)
- Power BI (Dashboard & Data Visualization)
- Google Colab
- Data Cleaning & Preprocessing

## 📊 Key Analysis Performed
- Data cleaning (handling missing values, duplicates)
- Feature engineering (date extraction: year, month)
- GroupBy and aggregation analysis (Python)
- Time-series analysis
- Interactive dashboard creation in Power BI:
  - KPI Cards (Total Sales, Profit, Orders)
  - Bar Chart (Sales by Region)
  - Pie Chart (Sales by Category)
  - Line Chart (Monthly Sales Trend)

## 📈 Key Insights
- Central region generated the highest sales
- Technology category contributed the most revenue
- Sales showed an increasing trend towards year-end
- Some products resulted in negative profit (loss)

## 📂 Dataset
- Superstore Sales Dataset (CSV format)

## 🚀 Project Outcome
Developed an interactive dashboard using Power BI and performed data analysis using Python to generate actionable business insights for decision-making.
